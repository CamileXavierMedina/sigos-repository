# SiGOS - Sistema de Gestão de Odontologia Social

Este repositório contém o protótipo funcional integrado e o portal de triagem socioeconômica do **SiGOS**, desenvolvido em parceria com a clínica Click Dent e a Inova Simples para as Atividades Curriculares de Extensão (ACE) e Projeto Integrador I do UniCEUB.

#  Como Visualizar o Protótipo Online

Para facilitar a avaliação da banca examinadora e do professor orientador, o sistema está hospedado e pode ser acessado de forma totalmente interativa diretamente pelo navegador, sem necessidade de instalação.
